## Target expansion shape

Caller writes:
```scheme
(:wat::kernel::run-threads
  (:wat::core::Tuple factory)
  client-fn)
```

Macro expands to (approximately):
```scheme
(:wat::core::let
  [thread       (:wat::kernel::spawn-thread
                  (:wat::core::fn
                    [server-rx <- :rust::crossbeam_channel::Receiver<I>
                     server-tx <- :rust::crossbeam_channel::Sender<O>]
                    -> :wat::core::nil
                    (factory (:wat::kernel::ThreadPeer/new server-rx server-tx))))
   client-peer  (:wat::kernel::ThreadPeer/new
                  (:wat::kernel::Thread/output thread)
                  (:wat::kernel::Thread/input  thread))
   result       (client-fn client-peer)
   _drained     (:wat::kernel::Thread/drain-and-join thread)]
  result)
```

Key shapes:
- The wrap-fn inside `spawn-thread` converts spawn-thread's raw `[rx, tx]` signature into the user-facing `ThreadPeer` shape via `ThreadPeer/new server-rx server-tx`. The user's factory writes against `ThreadPeer<I, O>`.
- Client-side peer constructed from `Thread/output` (Receiver<O>, parent reads what server wrote) + `Thread/input` (Sender<I>, parent writes what server reads).
- Server peer is `ThreadPeer<I, O>` (server reads I, writes O). Client peer is `ThreadPeer<O, I>` (client reads O, writes I). Same struct, mirror type-param binding per Stone C1 design.
- `Thread/drain-and-join` consumes the Thread<I,O> handle for clean shutdown.

## Macro shape (defmacro precedent at `wat/test.wat:228-231` and `wat/test.wat:780-790`)

Two macro precedents in the codebase:
- `:wat::test::program` (variadic `& (forms ...)` collector + `~@` splice)
- `:wat::test::run-thread` (single-arg wrap with quasiquote)

Both use `:wat::core::defmacro` at `:AST<wat::WatAST>` type level. D1's macro is a **2-arg defmacro** (factories-AST + client-fn-AST); it pattern-matches the factories-AST to extract the single factory child of the Tuple.

Quasiquote builds the expansion; `~` unquotes the factory AST and client-fn AST into the let-binding bodies.
